# Polza test task

Репозиторий с выполненными заданиями.

## Структура

- `task1_email_checker/` — скрипт для проверки email-доменов (Python + dnspython), внутри есть README по запуску.
- `task2_telegram_sender/` — скрипт, который отправляет текст из `.txt` файла в приватный Telegram-чат через бота (`send_to_telegram.py`).
- `task3_architecture/` — файл `architecture.md` с описанием архитектуры для 1200 email-адресов
